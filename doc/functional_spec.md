# 功能规格说明书 (Functional Specification)

**项目名称**: FlexTimer v12
**核心定位**: 演讲、会议、教学全场景智能计时专家

---

## 1. 软件概述
FlexTimer v12 是一款极致纯粹的桌面计时工具，采用 **玻璃拟态 (Glassmorphism)** 设计语言。它解决了传统计时器功能单一、界面生硬、无法灵活管理多个议程环节的痛点。

## 2. 核心功能集

### 2.1 议程链式管理 (Agenda Workflow)
- **动态录入**: 支持实时增加议程项，设置名称与计划金钟数。
- **流畅排序**: 无论在计时前还是计时中，用户均可随时通过拖拽调整议程顺序，系统自动重排任务链。
- **即时清理**: 快速重置任务列表，一键恢复初始状态。

### 2.2 极致计时体验 (Immersive Timing)
- **沉浸式准备**: 开始计时前提供 3-2-1 倒数遮罩，营造专业会议氛围。
- **超时保护与追踪**:
  - 倒计时结束后支持负向计时（Overtime），精确记录超长时长。
  - 红色高亮预警，强视觉反馈。
- **策略控制**: 支持一键跳过、暂停与继续，完美应对突发会场状况。

### 2.3 丰富的设置与交互 (Rich UI/UX)
- **全屏视角与极简模式**: 隐藏不必要的控制栏，只保留最核心的时间显示。
- **高度定制**: 用户可直接在界面上调整字体大小、不透明度及颜色。
- **交互细节**:
  - 侧边栏抽屉式切换，不挡主视觉。
  - 标准 Windows 右上角控制按钮，单手自如关闭。

### 2.4 历史与报告 (Data & Review)
- **多维历史回顾**: 集成日历视图，直观展示过去每一天的会议轨迹。
- **复盘报告导出**: 将今日所有议程、实际用时、超时数据导出为 CSV，用于会后分析。

## 3. 运行环境
- **操作系统**: Windows 10/11
- **硬件**: 占用极低，支持各类投影仪或显示分屏。
